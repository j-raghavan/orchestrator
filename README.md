Orchestrator is a Python package designed for simplified task orchestrations. Inspired by the `prefect` package, Orchestrator aims to provide a lightweight and user-friendly framework for defining, running, and managing workflows.

## Key Features

- **Flows:** Define workflows using a simple and intuitive decorator-based API.
- **Tasks:** Create reusable units of work with ease.
- **Deployment:** Manage and deploy your workflows effortlessly.
- **FastAPI Integration:** Enjoy a modern web-based UI for managing your orchestrations.

## Getting Started

### Installation

Install Orchestrator using pip:

```bash
pip install orchestrator
```
